# Wallet Service

## Overview

The Wallet Service manages user cryptocurrency wallets, balances, and transaction history. It provides secure storage and management of digital assets.

## Features

- Multi-cryptocurrency wallet support
- Balance management
- Deposit and withdrawal processing
- Transaction history tracking
- Wallet security and encryption

## Technology

- Spring Boot 3.2.0
- Java 21

## Running the Service

```bash
./gradlew :wallet:bootRun
```

## Testing

```bash
./gradlew :wallet:test
```

## Security

The wallet service implements industry-standard security practices including encryption, secure key management, and transaction verification.
