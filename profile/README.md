# Profile Service

## Overview

The Profile Service manages user accounts, authentication, and profile information for the exchange platform.

## Features

- User registration and authentication
- Profile management
- KYC (Know Your Customer) verification
- User preferences and settings
- Account security features
- Two-factor authentication (2FA)

## Technology

- Spring Boot 3.2.0
- Java 21

## Running the Service

```bash
./gradlew :profile:bootRun
```

## Testing

```bash
./gradlew :profile:test
```

## Security

The service implements secure authentication mechanisms and follows best practices for user data protection and privacy.
